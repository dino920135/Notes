- ## Pull Docker Image
	- ```bash
	  docker pull ros:melodic
	  ```
- ## Run Docker Container
	- ### First time
	  ```bash
	  docker run \
	  -it --detach \
	  --cpuset-cpus="0-23"
	  --net=host \
	  --volume /home/point001/TS:/home/TS \
	  --env ROS_MASTER_URI=http://localhost:11311 \
	  --name rosTS \
	  ros-melodic-tsradar
	  ```
	- ### Enter running container
	  ```bash
	  docker exec -it rosTS bash
	  ```
